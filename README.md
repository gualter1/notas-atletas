# Projeto devstart trilha 1

O projeto cria uma classe com os dados dos atletas, nessa classe é criado um constructor onde ele faz a filtragem dos dados eliminando as notas que não são validas e a partir desse contructor são criados os metodos.

* obterNome()
* obterPrimeiraNotaValida()

* obterUltimaNotaValida()
* obterMedia()


Ao final do projeto é devolvido uma string contendo todos esses dados em formato de texto.
